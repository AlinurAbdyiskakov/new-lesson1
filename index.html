Что такое HTML и CSS?

HTML (HyperText Markup Language) мазмундун структурасын жана анын маанисин, мисалы, аталыштар, абзацтар же сүрөттөр сыяктуу мазмунду аныктоо менен аныктайт. CSS (Cascading Style Sheets) же Cascading Style Sheets — бул, мисалы, шрифттерди же түстөрдү колдонуу менен мазмундун көрүнүшүн долбоорлоо үчүн арналган презентация тили. Бул эки тил, HTML жана CSS, бири-биринен көз каранды эмес жана ошол бойдон калууга тийиш. CSS HTML документинин ичинде жазылбашы керек жана тескерисинче. Эреже катары, HTML ар дайым мазмунду чагылдырат жана CSS ар дайым анын стилин аныктайт. HTML менен CSSтин ортосундагы айырманы түшүнүү менен, келгиле, HTMLге кененирээк киришели.
